# Auto Motion Art

Landing page automotiva moderna desenvolvida com React, Vite, TanStack Start e Tailwind CSS.

## Tecnologias

- React 19
- Vite
- TanStack Router / Start
- Tailwind CSS 4
- Motion
- TypeScript

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build de produção

```bash
npm run build
```

## Preview local

```bash
npm run preview
```

## Estrutura importante

### Configuração central

O arquivo abaixo centraliza os dados do site:

```text
src/config/site.ts
```

Nele você pode alterar:

- nome da empresa
- telefone
- WhatsApp
- Instagram
- endereço
- SEO
- slogan
- links do menu

## Componentes principais

- Navbar
- Hero
- CarShowcase
- Services
- Features
- Results
- Testimonials
- CTASection
- Footer

## Observações

O projeto ainda utiliza alguns dados demonstrativos.
Antes de publicar:

- substitua telefone
- substitua WhatsApp
- troque endereço
- troque Instagram
- revise textos e imagens

## Dependências possivelmente removíveis

O projeto possui bibliotecas que podem não estar sendo usadas atualmente, como:

- recharts
- react-day-picker
- input-otp
- react-resizable-panels
- alguns componentes Radix UI

Faça uma verificação antes de remover.
